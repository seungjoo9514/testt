DASS

종이문서의 엄청난 증가와 규제 정책에 따라 저장/관리/폐기를 위한 종이문서 분류와 관리가 수시로 일어 나고 있습니다. 이에 따라 문서 분류를 위한 수동 작업은 정보량이 많아질 수록 더욱 많은 인력과 시간이 소요되며 수작업에 따른 오류 발생 가능성 또한 높아집니다. 문서자동분류솔루션(DASS)은 대량의 종이문서 분류를 위한 자동화 솔루션으로서 종이 기록물을 효율적으로 관리하기 위해 적합한 범주를 구분하여 관리하고 처리된 결과를 사용자가 용이하게 검증 할 수 있도록 사용자 편의성을 제공합니다. 문서자동분류솔루션(DASS)은 초고속 문서분류기가 이미지 Data의 서식(Form)을 판별하고 서식(Form) 내에 포함된 텍스트 데이터 인식을 바탕으로 자동 분류한 종이문서 처리 결과를 솔루션이 제공하는 적합한 범주 내에서 추가정정 및 검증을 통해 더욱 완벽하게 분류될 수 있도록 다양한 편의 기능을 제공합니다.

제품 구성요소

효율적인 문서 분류 처리 지원

  • Document Classifier : 문서를 분류기준 정보에 의해 분류하고 얻어진 이미지 정보를 서버에 저장 하는 역할을 합니다.
  • Process Manager : 분류작업 대상 문서를 등록하여 문서가 분류 되어야 할 기준정보, 스캐너 정보, 작업자 등을 관리합니다.
  • Result Reviewer : 분류작업에 의해서 얻어진 이미지 정보를 확인하여 잘못된 정보를 교정 합니다. 이미지 정보 목록 보기와 이미지 보기 기능을 통해 이미지 정보를 쉽게 교정 할 수 있습니다.
  • Data Writer : 대용량 이미지 데이터 초고속 송수신 모듈로서 고속으로 처리되는 데이터의 안정성을 보장합니다.
  • Barcode Printer : 재처리 시 인식률 개선을 위한 정보 바코드 출력 및 Box 바코드 출력 모듈.
  • Inno Scanner : 초고속 분류기와 Interface를 용이하게 해 주는 통합 드라이버 모듈.

효율적인 문서 분류 처리 지원

DASS Classify(강력한 문서 분류 처리)

분류가 필요한 방대한 양의 문서를 사람이 직접 수동으로 분류하기 힘듭니다. 고속 분류 장비(550PPM)에 분류 기준정보만 입력하면 문서를 정해진 기준 데이터에 의해 분류 포켓으로 이동 시킵니다. 또한 분류하면서 얻어진 이미지 정보를 서버에 저장하여 분류 결과를 확인하고, 필요 시 오류를 보정할 수 있습니다.

DASS Classify?

혼재된 여러 서식에서 필요한 데이터만 추출하여 자동으로 판별 분류 해 줌으로써 문서 식별 및 분류에 투입되는 리소스와 시간을 효과적으로 절감할 수 있습니다. 문서 이미지에서 필요한 데이터만 추출하여 자동으로 서식의 종류 및 분류기준을 식별하고 분류 할 수 있습니다.

DASS Classify 주요 기능은 무엇입니까?

혼재된 여러 서식에서 필요한 데이터만 추출하여 문서 식별, 추출 데이터에 의해 자동 분류, 분류 이미지 정보를 DB화 하여 제공 합니다.

  • 각종 가입 신청서, 신고서, 설문지, 시험 답안지 같은 비교적 적은 양에서부터 방대한 양의 문서 처리에도 대응할 수 있도록 확장성을 고려하여 설계 되었습니다.
  • 방향/ 기울기/ 역상 보정 및 흑백 영상으로 이진화, 기밀 데이터 제거 등이 가능 합니다.

DASS Review(효율적인 문서분류 처리)

검증은 인식/분류 결과를 확인하여, 필요 시 오류를 보정하는 단계 입니다. 검증 속도를 높이고 절차를 단순화하기 위해 여러 가지 효과적이고 사용하기 용이한 User Interface를 제공합니다.

DASS Review?

속 장비를 통해 분류된 결과를 검증하고, 필요 시 오류를 보정하기 위한 처리 기능을 제공 합니다.

DASS Review의 주요 기능은 무엇입니까?

데이터 필드를 하나씩 확인 할 수 있는 목록 검증 기능을 제공하며, 썸네일 위주로 기준문서와 부가서류를 쉽게 구분 할 수 있습니다. 모든 데이터는 인식 결과와 원본 이미지를 나란히 보면서 확인 할 수 있고 오류 필드를 보정할 수 있는 기능 을 제공 합니다.

FEVER

FEVER : Rule기반의 전자문서 처리 솔루션

FEVER는 고객의 비즈니스 프로세스 자동화를 위해 온라인상의 문서(전자 서식, 데이터)에서 필요한 다양한 팩트(고객 정보, 유형, 상품정보, 기초 정보, 계약조건, 서비스 등)의 속성, 값을 Rule 기반으로 관리하고, 데이터에 대한 검증을 능동적으로 처리하기 위한 전자문서와 기간계간 데이터 처리 솔루션 입니다.

주요 서비스영역

업무의 기준이 되는 정책/규정, 법규, 업무절차 변경, 서비스확대에 유연하고 빠르게 대응하는 전자문서 정보관리 솔루션 입니다.

  • 온라인 계약시스템
  • 종이 없는 창구 시스템
  • 전자 교범 (전자업무 매뉴얼)
  • 코딩을 줄이고 운영효율성을 위한 rule 엔진
  • 기간계 인터페이스

주요 관리영역

전자문서에서 작성된 내용과 유관시스템과의 입출력 관계를 관리하여 업무처리 시스템과의 연계를 신속하게 적용 가능합니다.

Meta Management 업무규정, 서비스 확대와 같은 잦은 비즈니스 변화를 정책기반으로 서식, 증빙서류, 항목 검증 관계를 등록하고 입/출력 맵핑 정보를 통해 업무처리 시스템과 seamless하게 통합할 수 있습니다.
Meta Repository 비즈니스 정책, 서식 정보, 연계 시스템 정보가 보관되는 레파지토리 입니다.
Runtime API's 메타정보를 실시간 연동하여 서식적재, 이동, 항목 속성 변경, 이벤트 처리 대리자의 인터페이스를 제공하여 서식 솔루션에 관계없이 일관된 방법으로 처리 가능합니다.
Rule-Based Decision 비즈니스 상황을 탐지하여 서식, 입력항목, 유효성 검증 등에 필요한 의사결정 로직을 런타임 구성요소로 제공합니다.
I/O Mapper 업무처리 시스템과의 연계를 위한 입출력 정보를 제공합니다.
